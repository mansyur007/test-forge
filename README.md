# 🔧 TestForge

> **Free, open-source web-based test case management tool** — built for QA engineers who deserve better tooling without the price tag.

---

## ✨ What is TestForge?

TestForge is a lightweight, open-source alternative to tools like TestRail and Qase.io. It gives QA teams a clean, intuitive workspace to write, organize, and track test cases — without subscriptions, seat limits, or feature paywalls.

Whether you're a solo tester or part of a team, TestForge is built to fit into your workflow, not the other way around.

## 🎯 Planned Features

- 📁 Project and test suite organization
- ✅ Test case creation with steps, expected results, and priority
- 🔄 Test run management and execution tracking
- 📊 Basic reporting and pass/fail metrics
- 👥 Role-based access (Admin, QA Lead, Tester)
- 🔗 Integration-ready (CI/CD, Jira, Slack)
- 🌐 Self-hostable — own your data

---

## 🛠️ Tech Stack *(planned)*

- **Frontend:** React / Next.js
- **Backend:** Node.js / Express or Laravel
- **Database:** PostgreSQL
- **Auth:** JWT / OAuth2
